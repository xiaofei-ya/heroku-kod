# kodexplorer-heroku
# Deploying

Install the [Heroku Toolbelt](https://toolbelt.heroku.com/).

```sh
$ git clone git@github.com:oner12345/kodexplorer-heroku # or clone your own fork
$ cd kodexplorer-heroku
$ heroku create
$ git push heroku master
$ heroku open
```
